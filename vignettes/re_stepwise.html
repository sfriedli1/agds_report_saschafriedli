Description of the data set

The dataset consists of half-hourly measurements of ecosystem fluxes collected at various monitoring sites in Switzerland and is provided by FLUXNET. It contains about 23000 observations and 15 predictor variables, including radiation fluxes, temperature, humidity and air pressure. The target variable is GPP_NT_VUT_REF, which represents gross primary productivity (GPP) estimated using flux partitioning methods. Several variables are closely related or derived from each other, such as SW_IN_F and SW_IN_F_MDS (incoming shortwave radiation) or LW_IN_F and LW_IN_F_MDS (incoming longwave radiation). These interdependencies can lead to multicollinearity, which is important for model selection and interpretation.

Implementation of complete stepwise forward regression

# Preparation
target <- "GPP_NT_VUT_REF"
# Remove the siteid, because it is not numerical data
data_without_siteid <- raw_data %>%
  select(-siteid)

predictors <- setdiff(names(data_without_siteid), target)

current_predictors <- c()
remaining <- predictors

# Start only with Intercept
best_model <- lm(GPP_NT_VUT_REF ~ 1, data = data_without_siteid)
best_aic <- AIC(best_model)

stepwise_log <- list()

repeat {
  candidate_r2 <- list()

  # Fit all models with one additional predictor
  for (var in remaining) {
    formula <- as.formula(paste(target, "~", paste(c(current_predictors, var), collapse = " + ")))
    model <- lm(formula, data = data_without_siteid)
    r2 <- summary(model)$r.squared
    
    candidate_r2[[var]] <- list(
      model = model,
      r2 = r2
    )
  }

  # 2. Choose the model with best R2
  r2_values <- sapply(candidate_r2, function(x) x$r2)
  best_var <- names(which.max(r2_values))
  best_model_r2 <- candidate_r2[[best_var]]$model

  # 3. Calculate the AIC of the best model
  aic_new <- AIC(best_model_r2)

  # 4. Compare the AIC of the new model with the best model so far
  if (aic_new < best_aic) {
    best_aic <- aic_new
    current_predictors <- c(current_predictors, best_var)
    remaining <- setdiff(remaining, best_var)
    best_model <- best_model_r2

    # Safe the step
    stepwise_log[[length(current_predictors)]] <- list(
      predictor = best_var,
      R2 = summary(best_model)$r.squared,
      AIC = best_aic
    )
  } else {
    break  # If the AIC is worse quit the algorithm
  }
}

# Safe the final result in a data frame
stepwise_df <- tibble(
  step = seq_along(stepwise_log),
  predictor = sapply(stepwise_log, function(x) x$predictor),
  R2 = sapply(stepwise_log, function(x) x$R2),
  AIC = sapply(stepwise_log, function(x) x$AIC)
)

Analysis and discussion of the results

After going through the complete algorithm of the stepwise forward regression resulted in 10 of the 16 potential variables being included in the model. The early selection of PPFD_IN, LW_IN_F and VPD_F aligns well with known drivers of photosynthesis, highlighting the importance of long wave radiation for temperature regulation especially during the night and the vapor pressure deficit in regulating gross primary productivity. The visualization of the Modelcomplexity vs.AIC shows that for the first 5-6 Variable the AIC decreases significantly for each variable that is being added to the model. This is consistent with the R-Squared. According to the R-Squared the first Variable (PPFD_IN), which stands for the incoming photosynthetic photon flux density already explains 45.21 % of the variability in the GPP. For the first 5 additional variables the grafic shows that the R-Squared improves significantly. For the remaining variables in the model the improvements after adding an additional variable is less and less significant. Adding to many variables that only marginaly improve the R-Squared could lead the model to becoming prone to overfitting. Another interesting finding is that for example our model included LW_IN_F but it did not include LW_IN_F_MDS even though both variables stand for the longwave radiation. So we would expect that they would both explain a similar amount of the dispersion in the GPP. But although LW_IN_F is the variable that explains the second most variance of the GPP, LW_IN_F_MDS is not included in our model at all. The reason for this is probably that LW_IN_F is partly consolidated from LW_IN_F_MDS and therefore this two variables have a high multicollinearity. This is supported by the Pearson Correlation Coefficient of 0.985. In a regression model we should avoid to use variables that have a high multicollinearity due to its destabilizing effects on the predictions of the model. That is why only one of the two variables was included through the stepwise forward regression where as the other one was not. A similar explanation applies to SW_IN_F and SW_IN_F_MDS. These variable both stand for the incoming shortwave radiation and again SW_IN_F is partly consolidated from SW_IN_F_MDS. The final model explains 59.84% of the variance in GPP. This solid considering the complexity of ecosystem processes. However this also means that there are still ~40% that are not being explained with this model. In this stepwise forward regression we used a linear model. If we also captured non-linear interactions by using a non-linear model we possibly could explain more of the variance in the GPP.
